# Terminator Quick Guide

Terminator is a terminal emulator that lets you split the window into multiple terminals. This is useful in ROS because you often need more than one shell open at the same time.

## Why use Terminator for this exercise?

You can keep:

- one terminal in `ws_linux_command/src`
- one terminal in `my_bash_to_copy`

Then use `pwd` in each terminal to understand where you are and copy the file more easily.

## Useful shortcuts

`Ctrl + Shift + O` : Split horizontally.

`Ctrl + Shift + E` : Split vertically.

`Ctrl + Shift + T` : Open a new tab.

`Ctrl + Tab` : Move between terminals.

`Ctrl + Shift + W` : Close the current terminal.
